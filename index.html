Get style + behavior.

WireUI is a zero-config UI library. Drop in the CDN, write semantic class names, and your inputs, forms, and widgets are automatically styled and wired up — no extra JavaScript needed.

📦 Add to your HTML

<!-- WireUI CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/wireui/index.css" />

<!-- WireUI JS (before </body>) -->
<script src="https://cdn.jsdelivr.net/npm/wireui/index.js"></script>

That's it. No npm install, no build step, no config. Just write class="ui-*" and it works.

JS API
WireUI exposes a global WireUI object for programmatic control.
// Show a toast notification
WireUI.toast('Message here', 'success'); // success | error | warning | info
WireUI.toast('Saved!', 'success', 4000); // custom duration (ms)

// Open / close a modal
WireUI.openModal('#myModal');
WireUI.closeModal('#myModal');

// Set a progress bar value (0–100)
WireUI.setProgress('#myProgress', 75);
